# led-pomodoro
A small addressable LED project with Plasma 2040 and Neopixels. Implements Pomodoro technique with LEDs (https://en.wikipedia.org/wiki/Pomodoro_Technique).

Followed this tutorial to get the LEDs working: https://learn.pimoroni.com/article/plasma-2040

Editor used: Thonny
Running MicroPython (for now at least)
